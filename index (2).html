AWS security
assessments for robust cloud compliance

GHIDOR is an advanced automated audit tool designed to help security teams and cloud engineers evaluate and improve the security posture of their AWS infrastructure. By leveraging the CIS AWS Foundations Benchmark (Levels 1 & 2), GHIDOR provides actionable recommendations, detects risky configurations, and generates interactive reports (HTML & JSON).
