CIS Benchmark

Le CIS (Center for Internet Security) Benchmark fournit des lignes directrices pour sécuriser les configurations cloud.

  • Level 1 : Contrôles de base recommandés pour tous les environnements, facile à mettre en œuvre sans impact opérationnel majeur.
  • Level 2 : Contrôles avancés pour les environnements nécessitant une sécurité renforcée, pouvant exiger des ajustements opérationnels.

Ces benchmarks servent de référence pour les audits de conformité sur les services comme IAM, S3, RDS, EC2, CloudTrail, etc.
